#### 笔记

根据时间，可以LucaSystem的脚本类型分为三个版本，目前仅研究V3版本，即最新版本。LucaSystemTools支持V2版本的脚本解析

| 类型  |  长度 | 名称 | 说明                                 | 
|-----|-------|-----|------------------------------------|
| uint16 |  2  | len | 代码长度                               |
| uint8 |  1  | opcode | 指令索引                               |
| uint8 |  1  | flag | 一个标志，值0~3                          |
| []uint16 |  2 * n  | data0 | 未知参数，其中n=flag(flag<3),n=2(flag==3) |
| params |  len -4 -2*n  | params | 参数                                 |
| uint8 |  k  | align | 补齐位，其中k=len%2                      |
